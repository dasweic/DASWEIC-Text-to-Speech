# Multi-Language Text-to-Speech Converter (Python)

A simple and efficient Python-based Text-to-Speech (TTS) application that converts written text into natural-sounding speech in multiple languages using Google Text-to-Speech (gTTS).

---

## Overview

This project provides a command-line interface for converting text into speech across more than 30 supported languages. It is designed to be lightweight, easy to use, and suitable for beginners as well as developers who want a quick text-to-speech solution.

---

## Features

- Converts text into high-quality speech
- Supports more than 30 languages
- Automatically generates MP3 audio files
- Simple and clean command-line interface
- Built using Python and Google Text-to-Speech (gTTS)
